## 👨‍💻 About

This is my personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies. It showcases my projects, skills, experience, and resume as an embedded systems engineer.

---

## ✨ Features

- ⚡ **Loading animation** — Circuit-ring loader on first visit
- 🌗 **Light / Dark theme toggle** — Smooth transition with persistent state
- 🖱️ **Interactive grid background** — Responds to cursor movement with glowing intersections
- 🔬 **Animated circuit SVG** — Hero section PCB schematic with signal trace animations
- 🔤 **Logo cycling** — Hover `ALBIN` in the nav to cycle `.ENGINEER` → `.DESIGNER` → `.TRADER` → `.SOLVER`
- 📱 **Fully responsive** — Mobile hamburger menu, tablet and desktop layouts
- 🎨 **Scroll reveal animations** — Sections animate in as you scroll
- 📄 **Resume download** — One-click PDF download from the hero CTA
- 🔗 **Social links** — LinkedIn, Gmail, X (Twitter)
